== Description ==

Woocommerce order action email allow to add multiple custom order action and send email to customers.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/woo-order-action-email` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add Multiple Email Woocommerce->Action Emails /wp-admin/edit.php?post_type=woae_email
